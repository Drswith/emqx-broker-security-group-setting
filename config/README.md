#### 使用说明

1.  按照需要的厂商，点击对应文件夹进入

    | 文件夹路径    | 公有云厂商 |
    | ------------- | ---------- |
    | aliyun        | 阿里云     |
    | baidu_cloud   | 百度云     |
    | huawei_cloud  | 华为云     |
    | tencent_cloud | 腾讯云     |

2.  登录目标公有云服务器控制台
3.  在控制台中，进入安全组设置页面（不同服务商有不同的入口）
4.  导入目标安全组配置文件即可
