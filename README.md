# EMQ X 安全组预置配置

#### 仓库介绍

在公有云服务器上安装 EMQ X Broker 时,需要开放 EMQ X Broker 默认工作端口，本仓库为各大公有云服务器安全组设置配置文件集合

#### 使用说明

1.  登录目标公有云服务器控制台
2.  在控制台中，进入安全组设置页面（不同服务商有不同的入口）
3.  导入目标安全组配置文件即可
